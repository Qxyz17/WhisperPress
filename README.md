# WhisperPress

WhisperPress 是一个注重隐私和安全的聊天应用，采用端到端加密（E2EE）技术，确保用户之间的消息只能被发送者和接收者读取。该项目旨在提供一个私密、安全、简洁的沟通平台，保护用户的通信内容免受外部窥探。

## 特性

* **端到端加密**：所有消息都通过端到端加密进行保护，确保消息只能被发送方和接收方访问。
* **开源**：WhisperPress 是一个完全开源的项目，欢迎任何开发者参与贡献。
* **简单易用**：界面简洁，操作直观，专注于核心功能。
* **跨平台支持**：目前支持 Web 和移动端（正在开发中）。

## 安装

### 前提条件

* Node.js（版本 14 或以上）
* npm（或 yarn）

### 安装步骤

1. 克隆项目到本地：

   ```bash
   git clone https://github.com/Qxyz17/WhisperPress.git
   cd WhisperPress
   ```

2. 安装依赖：

   ```bash
   npm install
   # 或者使用 yarn
   yarn install
   ```

3. 启动开发环境：

   ```bash
   npm start
   # 或者使用 yarn
   yarn start
   ```

4. 在浏览器中访问：

   ```
   http://localhost:3000
   ```

## 使用

1. 注册账户并登录
2. 创建或加入聊天房间
3. 开始进行私密对话，所有聊天内容都将自动加密
